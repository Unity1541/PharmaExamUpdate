# 線上考試系統 - 設計改造紀錄

## 改造概述

已將 `index.html` 的使用者登入畫面和儀表板從基礎樣式改造為 `code.html` 的現代化 Tailwind CSS 設計風格，並加入完整的儀表板功能。

## 主要改動

### 1. **HTML 結構更新** (`index.html`)

- ✅ 整合 Tailwind CSS 框架（來自 CDN）
- ✅ 添加自定義顏色主題配置
- ✅ 使用 Material Symbols 圖標
- ✅ 應用玻璃態效果 (glass-panel)
- ✅ 改進的響應式設計（支援手機、平板、桌面）
- ✅ 現代化的加載指示器動畫

### 2. **登入畫面** (`createLoginViewHTML()`)

採用 `code.html` 的設計風格：

- 🎨 奶油色背景 + 漂亮的背景裝飾
- 📱 居中的卡片布局（最寬 448px）
- 🎯 學生/管理員角色切換器（Material Symbols 圖標）
- 💫 動畫效果：懸停時按鈕升起、背景裝飾變大
- 🔐 改進的輸入框樣式（焦點時發光）
- 🎨 配色：奶油色、桃紅色、咖啡色、鼠尾草綠

**新配色方案：**

- 主色：`#ffaa8e` (桃紅色)
- 次色：`#b5d6b0` (鼠尾草綠)
- 背景：`#fffcf7` (奶油色)
- 文字：`#5c4b43` (咖啡色)

### 3. **儀表板** (`createStudentDashboardViewHTML()`)

完整重設計，採用 code.html 的多卡片布局：

#### 統計卡片區 (4 個卡片)

- 已測驗次數 (鼠尾草綠主題)
- 平均分數 (天藍色主題 + 進度條)
- 收藏題目數 (桃紅色主題)
- 學習狀況 (紫色主題)

#### 左欄 (5/12 寬度)

1. **快速開始卡片**

   - 大按鈕直接開始新測驗
   - 漸變背景 (桃紅色)

2. **成就卡片**
   - 根據測驗次數和分數顯示徽章
   - 陽光黃色背景

#### 右欄 (7/12 寬度)

1. **最近測驗卡片**

   - 顯示最近 5 場測驗
   - 得分、科目、日期顯示
   - 查看詳情按鈕

2. **收藏題目卡片**
   - 列出所有收藏的題目
   - 支援刪除和快速查看

### 4. **側邊欄** (`createSidebarHTML()`)

完全使用新設計：

- 📌 頂部 Logo + 品牌名稱
- 🔗 導航菜單（根據角色顯示不同選項）
- 👤 用戶資料卡片 (底部)

- ✨ 懸停效果、活動狀態指示

### 5. **Header 改進**

- ☀️ 動畫太陽圖標
- 🕐 實時時鐘（每秒更新）

### 6. **JavaScript 更新** (`index.js`)

- ✅ `createLoginViewHTML()` - 新登入畫面 HTML

## 設計特點

### 配色系統

| 顏色名稱     | 代碼      | 用途         |
| ------------ | --------- | ------------ |
| Cream        | `#fffcf7` | 背景         |
| Cream Dark   | `#f0e6da` | 邊框、分隔符 |
| Peach        | `#ffaa8e` | 主色、按鈕   |
| Peach Hover  | `#e67a5b` | 懸停狀態     |
| Coffee       | `#5c4b43` | 主文字       |
| Coffee Light | `#8d7a71` | 副文字       |
| Sage         | `#b5d6b0` | 次色         |
| Sage Dark    | `#88a684` | 次色深       |
| Sky          | `#b8dbe8` | 資訊色       |
| Lavender     | `#d8c2e6` | 紫色主題     |
| Sun          | `#ffd978` | 強調色       |

### 圓角設計

- `rounded-xl` (1rem) - 小元素
- `rounded-2xl` (1.5rem) - 中等元素
- `rounded-3xl` (2rem) - 大卡片
- `rounded-4xl` (2.5rem) - 最大卡片

### 陰影系統

- `shadow-soft` - 柔和陰影（背景）
- `shadow-card` - 卡片陰影
- `shadow-float` - 浮動效果（懸停時）
- `shadow-inner-soft` - 內陰影

### 字體

- 全站使用 `Quicksand` 字體
- 權重：400, 500, 600, 700

## 響應式設計

- 📱 手機：單欄布局
- 📱 平板：2-3 欄
- 🖥️ 桌面：完整 4+ 欄布局
- 📊 側邊欄在小屏幕隱藏 (hidden md:flex)

## 互動效果

- ✨ 卡片懸停：陰影增大、色彩變化
- 🔄 按鈕懸停：顏色漸變、上升效果 (-translate-y-0.5)
- ⏸️ 轉換動畫：200-500ms 平滑過渡
- 🌀 加載指示器：旋轉動畫
